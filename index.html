iBanda

iBand is a web application for a digital musical repository and their scores. It allows to securely archive and access all band information. The main features include consultation and management of music, visualization of musical scores, event management, news, users, repertoire, and a small encyclopedia of stored material. The development followed the MVC standard. It is also possible to catalog events and introduce news using attribute grammars.
